**Tagline**: AI-Powered Dubai Real Estate Insights

**Font Family**: Plus Jakarta Sans (Google Fonts)

### Type Scale
- H1: 60px (Bold for emphasis)
- H2: 40px
- T1: 24px (Bold for emphasis)
- T2: 22px (Bold for emphasis)

- B2: 14px

## Key Features

### Dashboard Metrics
- Y-o-Y (Year-over-Year) growth indicators
- Q-o-Q (Quarter-over-Quarter) growth indicators
- Future value prediction graphs
- Area heatmaps for property values

### Property Cards
- Property image with aspect ratio 16:9
- Price in AED with Y-o-Y growth indicator
- Location with Dubai area name
- Key metrics: bedrooms, bathrooms, sqft
- Future value prediction badge

### Charts & Graphs
- Line charts for price trends
- Bar charts for area comparisons
- Heatmap for geographic value distribution
- Growth indicators with color coding (green for positive, red for negative)

### Layout
- Clean, minimal design
- Professional appearance for financial decisions
- Responsive grid layouts
- Consistent spacing using 4px grid system
